# Voxel-engine

Voxel-engine is a project to create a voxel map. Some planets composed with cubes, where we can move.

## Dependencies

- `glfw3`
- `glew`
- `pkg-config`

## Instruction

To compile
```
$ make
```

To clean compilation files
```
$ make clean
```

To clean build
```
$ make fclean
```
